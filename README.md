# 🚨 Emergency Service Directory

A demo emergency service web application that allows users to quickly access important hotline numbers and simulate calling essential services.

---

## 🌐 Live Demo
👉 [View Live Project](your-live-link-here)

---

## 🚀 Features
- View emergency service cards (Police, Fire, Ambulance, etc.)
- Call functionality with coin deduction system
- Copy hotline numbers to clipboard
- Add to call history with current time
- Clear call history
- Real-time UI updates (coin, copy count, heart count)

---

## 🔧 Technologies Used
- HTML
- Tailwind CSS
- DaisyUI
- JavaScript (DOM Manipulation)

---

## 📌 Project Purpose
This project was built to practice JavaScript DOM manipulation and build interactive UI using modern frontend tools.

---

## ⚠️ Usage Note
- Each call will deduct 20 coins
- If coins are less than 20, call will be blocked
- Copy button will increase copy count
- Heart icon increases favourite count
